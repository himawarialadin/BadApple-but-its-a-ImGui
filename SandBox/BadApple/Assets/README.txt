---Bad Apple!!の動画と音声について---

BadApple.mp4 : 
以下のURLからBad Apple!!の動画をダウンロードし、「BadApple.mp4」として保存してください。
https://www.youtube.com/watch?v=FtutLA63Cp8

BadApple.wav : 
BadApple.mp4から音声を抽出し、WAV形式で「BadApple.wav」として保存してください。

ffmpegが使える環境の場合は、以下のコマンドを実行することで抽出できます。
ffmpeg -i BadApple.mp4 -vn -acodec pcm_s16le -ar 44100 -ac 2 BadApple.wav
